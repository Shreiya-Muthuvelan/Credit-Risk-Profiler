# 💳 Credit Risk Profiler 🏦

An intelligent neural network-based system that predicts the **credit risk level** (Low, Medium, High) of individuals based on financial and behavioral factors.

## 🚀 Live Demo

🔗 Try the app here: [https://subtle-sign-detector-ka9g5apjnjvkmvufuy8jow.streamlit.app/](https://credit-risk-profiler-ecaxbstztixgzhyxbe9xhv.streamlit.app/)


## 📌 Project Overview

This project builds and deploys a **multi-class classification model** to predict the credit worthiness of individuals using historical financial and credit-related data.  
It is designed to assist banks and financial institutions in quickly profiling clients and making data-driven decisions.

---

## 🧠 Key Features

- 🧮 **Neural Network Classification Model** using TensorFlow/Keras  
- ⚖️ Tackles **class imbalance** using SMOTE  
- 🧼 Handles **missing values, scaling**, and preprocessing  
- 🎯 Predicts 3 classes: **Low**, **Medium**, and **High** credit risk  
- 🖥️ **Streamlit Web App** with an intuitive banking-themed UI  
- 📊 Model Evaluation: Precision, Recall, F1-score
